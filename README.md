# uri-desafios

## 🧩 Beecrowd (URI Online Judge) Solutions - Java

Este repositório contém minhas resoluções dos desafios da plataforma Beecrowd.
O objetivo principal é o aprimoramento contínuo em Lógica de Programação, Estruturas de Controle, Manipulação de Dados e Complexidade Algorítmica utilizando Java.

---

## 📌 Sobre a Plataforma

A Beecrowd é um dos maiores portais de desafios de programação do mundo.
A resolução desses problemas exige:

- Domínio da sintaxe Java
- Raciocínio lógico aplicado a problemas matemáticos e sequenciais
- Controle de fluxo (if, while, for)
- Atenção à performance (limites de tempo e memória)
- Precisão na formatação de saída conforme especificação

---

## 🛠️ Tecnologias Utilizadas

- Linguagem: Java 25+
- Entrada e Saída: Uso da classe `Scanner` para leitura de dados
- Padronização internacional com `Locale`
- Implementações considerando restrições de tempo de execução e consumo de memória

---

## 🎯 Objetivo

Fortalecer a base em algoritmos e lógica de programação, visando evolução técnica contínua e preparação para desafios reais de desenvolvimento de software.
